# NextOn Theme — Custom Storefront

A modern, handcrafted Shopify theme built on top of the **Shopify Skeleton Theme** — not Dawn, not Horizon. Designed pixel-by-pixel from a [Nexton fashion - Mobile/web](https://www.figma.com/community/file/1365270188947832703), and implemented with **Tailwind CSS v4**, **Vanilla JavaScript**, and native **Web Components**.

---

## 🔗 Live Preview

> **Store URL:** [https://nexton-demo.myshopify.com](https://nexton-demo.myshopify.com) **Store Password:** `tahir`

---

## ✨ Features

- **Web Components** — UI elements are encapsulated as reusable, framework-agnostic native web components
- **Tailwind CSS v4** — Utility-first styling with the latest Tailwind CSS release
- **Fully Responsive** — Mobile-first layout that adapts gracefully to all screen sizes
- **Vanilla JavaScript** — No heavy frameworks; lean and fast by design
- **Shopify Skeleton Base** — Built on the minimal Shopify Skeleton Theme, not Dawn or Horizon
- **Figma-to-Code** — Faithfully implemented from a community Figma design system
